This project is built using a modern full-stack TypeScript architecture consisting of the following key technologies and frameworks:
1. Frontend Layer

* React 19 (SPA): Powers the user interface, utilizing modern functional components, state management (useState, useEffect), and context boundaries.
* Vite 6: Functions as the lightning-fast frontend bandler and development engine, integrating smoothly with the React applet framework.
* Lucide React: Provides the vector iconography suite used throughout the interface.
* Motion: Handles interactive animations and transitions (importing from motion/react).
2. Backend Server Layer

* Node.js (TypeScript Runtime): Consolidates server execution using Node's modern ecosystem capabilities.
* Express 4: Serves as the web framework to handle RESTful API endpoints (such as ticket CRUD operations, SLA triggers, user management, and AI data parsing).
3. Database / Persistence Layer

* JSON File Storage (fs / path): Utilizes a server-side JSON-based flat-file database system (data/db.json) for data persistence. This approach acts as a lightweight document-store database, complete with pre-seeded data, automated validation, and atomic writes.
4. AI & Intelligent Automation Layer

* Google Gemini SDK (@google/genai): Integrates the advanced @google/genai TypeScript SDK server-side (using models like gemini-3.5-flash) to handle smart ticketing functions, build-checks, risk scoring, category deduction, and generating detailed executive summaries.
5. Styling & User Interface

* Tailwind CSS v4: Serves as the CSS styling engine, compiling directly through the @tailwindcss/vite plugin.
* Modern Custom Typography: Configured with sleek, high-contrast typography pairings (such as "Inter" for interface controls and monospace accents).
6. Build System & Tooling

* TypeScript 5: Ensures robust type safety, contracts, and modular validation across the codebase.
* esbuild: Used in the build phase to bundle the backend server.ts into a standalone, portable CommonJS bundle (dist/server.cjs) to guarantee reliable deployment behavior.
* tsx: Executes TypeScript server files directly in the development workspace.
